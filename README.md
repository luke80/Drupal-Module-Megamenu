BYU Megamenu Module
======================
Drupal module enabling mega menu functionality for BYU theme

Installation Instructions
-------------------------
1. Download the module
2. Unpack into your sites/all/modules directory
3. Enable the module in Drupal (Admin > Modules)
4. Run the Drupal database update by navigating to yourSite/update.php
5. Settings can be accessed by going to 'structure > menus'. Click on 'list links', then edit the specific link.

### Notes:
*Parent links should be marked with 'Show as expanded.'
*Parent should not have any megamenu options selected.
*Children links can use feature or picture links.
